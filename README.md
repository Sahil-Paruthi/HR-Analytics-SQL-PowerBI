# 📊 HR Analytics Dashboard | SQL + PostgreSQL + Power BI

## 📌 Project Overview

This project is an end-to-end HR Analytics solution built using **PostgreSQL**, **SQL**, and **Power BI**. The objective is to analyze employee data, uncover workforce insights, and present key HR metrics through an interactive dashboard.

The project demonstrates SQL querying, data analysis, DAX calculations, and dashboard development using business intelligence best practices.

---

## 🎯 Project Objectives

- Analyze employee workforce data.
- Identify department-wise employee distribution.
- Evaluate average salary across departments.
- Measure employee attrition and promotion rates.
- Track hiring trends over time.
- Analyze workforce demographics.
- Build an interactive HR dashboard for decision-making.
